# 星尘快递

一个浏览器街机原型：驾驶快递飞船穿过碎片航道，采集星尘，装满货舱后返回右下角轨道港交付。每次交付都会进入更高威胁航段。

## 运行

直接打开 `index.html` 即可游玩。

也可以在项目目录运行：

```bash
python3 -m http.server 8899 --bind 127.0.0.1
```

然后访问 `http://127.0.0.1:8899/`。

## 手机 App / PWA

当前网页已支持手机安装版：

- Android Chrome：打开 `http://127.0.0.1:8899/` 或部署后的 HTTPS 地址，选择“添加到主屏幕”
- iPhone Safari：打开页面，选择分享按钮里的“添加到主屏幕”
- 已加入 `manifest.webmanifest`、`sw.js`、App 图标和离线缓存基础资源

## 微信小程序

微信小程序版本在 `wechat-miniprogram/`：

- 打开微信开发者工具
- 选择“导入项目”
- 项目目录选择 `wechat-miniprogram`
- AppID 可先使用测试号或 `touristappid`
- 入口页面是 `pages/game/game`

小程序版使用 Canvas 重建了手机触控循环：左侧拖动飞行，右侧按住推进，保留采集、交付、改装、连击奇星、红警飞机和地雷惩罚。

## 控制

- `WASD` 或方向键移动
- `Space` 推进
- `P` 暂停
- `Enter` 开始
- 停靠改装时按 `1` / `2` / `3` 选择
- 失败后按 `R` 重开
- 触屏设备使用左侧摇杆和右侧推进按钮

## 排行榜

- 游戏结束后可以输入昵称保存成绩
- 排行榜保存当前浏览器的前 10 名
- 排名按分数排序，同分时优先航段和交付数
- 结算页可以清空本地排行榜

## 当前循环

- 采集星尘，填满货舱
- 连续采集会积累连击倍率，受击或交付会清空连击
- 3 连击后会触发随机奇星机制，可能出现加分、充能、护盾、货舱或回声星尘奖励
- 每段航线都会生成一个随机合约，完成交付时可获得额外奖金
- 返回轨道港完成交付
- 交付后选择一项停靠改装，再进入下一航段
- 改装会改变主机外观：推进喷口、护盾环、回收磁场、反应堆核心和货舱模块都会显示在飞船上
- 能量消耗速度提升 20%，主机颜色会随能量下降切换，低于 30% 时进入红色警戒
- 随航段增加小行星和追踪无人机；撞到小行星后可能触发红警飞机或航道地雷
- 能量或护盾归零时本轮结束

## 已加入的反馈

- 屏幕边缘目标箭头会指向最近星尘、低能量时的能量核心，或满载后的轨道港
- HUD 目标行会显示当前合约状态和连击数
- 桌面视口右侧有雷达，显示玩家、星尘、能量核心、轨道港和追踪无人机
- 雷达和目标箭头会提示随机奇星；底部 HUD 会显示当前改装总阶数和各模块 Lv
- 采集、受击、交付和改装会在首次开始游戏后触发轻量音效
- 改装可叠加，影响推进、护盾、拾取范围、能量上限和交付需求
- 顶部和底部 HUD 已改成透明框，并会在飞船靠近时自动淡出避让
